# v0.1 OpenAI-Compatible Server

## Goal

Expose the v0.0 generator through OpenAI-compatible HTTP endpoints.

## Why

Production serving starts with a stable API that existing tools can call.

## Scope

- FastAPI server
- `/v1/models`
- `/v1/completions`
- `/v1/chat/completions`
- SSE streaming

## Out Of Scope

- continuous batching
- auth
- multi-model routing

## Acceptance Criteria

- `curl /v1/chat/completions` returns a valid response.
- Streaming emits incremental tokens.
- Basic request/response protocol types are covered by tests.
