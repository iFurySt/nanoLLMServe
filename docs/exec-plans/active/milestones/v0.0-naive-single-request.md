# v0.0 Naive Single Request

## Goal

Load a Hugging Face causal LM and generate tokens for one prompt.

## Why

Create the smallest working loop before adding serving complexity.

## Scope

- `AutoModelForCausalLM`
- tokenizer
- greedy sampling
- temperature sampling
- CLI generate command

## Out Of Scope

- HTTP API
- batching
- KV cache optimization beyond what the model exposes by default

## Acceptance Criteria

- A real small model can generate text from a CLI prompt.
- The demo works from a clean checkout.
- The README points to this milestone as the first step.

## What This Version Teaches

This milestone shows the smallest useful inference loop:

1. Load a causal LM and tokenizer.
2. Encode one prompt into token IDs.
3. Run the model over the full growing sequence.
4. Select the next token with greedy or temperature sampling.
5. Append the token, stop at EOS or `max_new_tokens`, and decode the generated
   tokens back to text.

The loop intentionally recomputes the full sequence each step. That makes the
baseline easy to inspect and gives later KV cache and batching milestones a
clear performance problem to solve.
