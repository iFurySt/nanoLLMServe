# v0.0 Naive Single Request

## Goal

Load a Hugging Face causal LM and generate tokens for one prompt.

## Why

Create the smallest working loop before adding serving complexity.

## Scope

- `AutoModelForCausalLM`
- tokenizer
- greedy sampling
- temperature sampling
- CLI generate command

## Out Of Scope

- HTTP API
- batching
- KV cache optimization beyond what the model exposes by default

## Acceptance Criteria

- A real small model can generate text from a CLI prompt.
- The demo works from a clean checkout.
- The README points to this milestone as the first step.
