# v0.9 Structured Output

## Goal

Support constrained decoding for JSON or schema-shaped output.

## Why

Production agent systems often need machine-readable outputs, not only free-form chat.

## Scope

- regex or JSON grammar
- token mask
- guided decoding path
- request-level structured output option

## Out Of Scope

- every JSON Schema feature
- external grammar engines unless the simple path becomes too noisy

## Acceptance Criteria

- A request can ask for JSON-shaped output.
- Invalid next tokens are masked during decoding.
- Tests cover at least one valid and one invalid path.
