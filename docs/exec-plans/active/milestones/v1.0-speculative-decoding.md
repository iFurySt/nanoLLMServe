# v1.0 Speculative Decoding

## Goal

Use a draft model to propose tokens and a target model to verify them.

## Why

Speculative decoding is a major low-latency optimization for modern LLM serving.

## Scope

- tiny draft model
- draft N tokens
- target verification
- accept/reject logic
- benchmark acceptance rate and latency

## Out Of Scope

- multi-draft strategies
- production-grade scheduling interactions
- custom kernels

## Acceptance Criteria

- Draft and target models can run in one request path.
- Accepted tokens are emitted correctly.
- Benchmark shows acceptance rate and latency impact.
